# kc-led-life-predict-core

Engineering-grade LED lumen maintenance life projection tool  
Based on LM-80 test data and TM-21 projection method.

## What this is
- Pure front-end (HTML / JS)
- No server, no backend
- Fully auditable calculation
- TM-21 cap enforced

## What this is NOT
- Not BLU or system lifetime
- Not marketing life claim
- Not a black box

## How to use
1. Enter LM-80 data (flux or relative R)
2. Set sample size, test hours, fit mode
3. Calculate
4. Use only reported values (cap applied)
